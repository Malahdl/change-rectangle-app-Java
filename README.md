# Welcome to the Change-Rectangle app!

# Description:
This is a simple JavaFX app where you can change the shape of a rectangle.

# Features:
1. You can change the height and width of the rectangle.
2. You can change the color of the rectangle.
3. You can change the stroke width of the rectangle.

# Important Notes:
* You can watch the demonstration video from the "demonstration" folder for more clarity.
* You can see some snapshots of the app in the "samples" folder.
